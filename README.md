🏦 Bank Management System
Overview

This is a simple Bank Management System built using Java technologies. The project demonstrates the implementation of authentication, database connectivity, and basic banking operations using JSP, Servlets, JDBC, and MySQL.

Features
User Registration
User Login Authentication
Secure Session Management
Deposit Money
Withdraw Money
Transfer Money
Check Account Balance
Responsive User Interface
